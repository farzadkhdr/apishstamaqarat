# API بۆ سیستەمی داواکاریەکان

ئەم APIیە بۆ پەیوەندی نێوان دوو سیستەمە:
1. سیستەمی ناردنی داواکاری (`https://dawakrdn.vercel.app`)
2. سیستەمی بەڕێوەبردنی داواکاریەکان (`https://systamwargrtn.vercel.app`)

## ✨ تایبەتمەندیەکان

### 🔒 داتا پارێزراوە
- **هەموو داتاکان لە JSON فایلەکان پارێزراون** و لەناو نانرێن
- **کاتێ رەتکردنەوە دەکەیت**، داتاکان دەمێننەوە لە `requests.json` بە دۆخی `rejected`
- **تەنها بەڕێوەبەر دەتوانێت داتا بسڕێتەوە** بە بەکارهێنانی `x-admin-token`
- **داتای نمونە** بەخۆکار دروست دەبێت بۆ یەکەمجار

### 📊 سیستەمی بەڕێوەبردنی تەواو
- وەرگرتنی داواکاریەکان
- نوێکردنەوەی دۆخ (وەرگرتن، ڕەتکردنەوە، لە کاردایە)
- ئاماری ڕاستەقینە
- فیلتەرکردن و ڕێزبەندی

### 📢 سیستمێکی رێکلامی پێشکەوتوو
- ناردنی رێکلامی نوێ
- ئەپلۆدی وێنە
- شوێنەواری بینین و کلیک
- چالاک/ناچالاک کردنی رێکلامەکان

### 🏠 بەڕێوەبردنی خانوو و زەوی
- زیادکردنی خانوو و زەوی
- نوێکردنەوەی دۆخی فرۆشتن
- پەیوەندی بە داواکاریەوە
