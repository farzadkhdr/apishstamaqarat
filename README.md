# سیستەمی API بۆ داواکاریەکان

APIی پێشکەوتوو بۆ سیستەمی ناردنی داواکاری و بەڕێوەبردنی داواکاریەکان

## ✨ تایبەتمەندیەکان

- ✅ **داتا پارێزراوە** لە JSON فایلەکان
- 🔒 **سیستەمی پاراستن** بە Tokenی ئەدمین
- 📊 **ئاماری تەواو** و ڕاپۆرت
- 💾 **پاشەکەوتی خۆکار** هەر کاتژمێرێک
- 🚀 **خێرایی بەرز** و جێگیری
- 🌐 **پاڵپشتی CORS** بۆ هەموو دۆمەینەکان

## 🚀 دەستپێکردن

### پێویستیەکان
- Node.js 16+ 
- npm 8+

### نەخشەکان
```bash
# کلۆنکردنی پرۆژە
git clone https://github.com/yourusername/request-system-api.git
cd request-system-api

# دانانی پاکەتەکان
npm install

# دروستکردنی فایلەکانی ڕێکخستن
cp .env.example .env

# دەستپێکردنی سێرڤەر
npm run dev  # بۆ گەشەپێدان
# یان
npm start    # بۆ جێبەجێکردنی ئاسایی
